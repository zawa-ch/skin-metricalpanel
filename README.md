# skin-MetricalPanel

AIMPのためのスタイリッシュなスキン  
その昔に自分用に制作し、公開していたもの  

残っていたデータを基にリポジトリとして構築したものであるため、情報が抜け落ちている部分があります  
ご了承ください  

## 動作環境

AIMP for Windows v4.10、またはそれ以降

AIMP公式サイトから最新のバージョンをダウンロードできます: [http://www.aimp.ru/](http://www.aimp.ru/?do=download&os=windows)
